# How to View the Logo Carousel

## Option 1: Vite Dev Server (Recommended - Full React App)

```bash
cd /Users/mattsilverman/Experiments_Logo-Carousel
npm install
npm run dev
```

Then open the URL shown in terminal (usually `http://localhost:5173`)

## Option 2: Python HTTP Server (Quick Preview)

```bash
cd /Users/mattsilverman/Experiments_Logo-Carousel
python3 -m http.server 8000
```

Then open: `http://localhost:8000/preview.html`

## Option 3: Direct File (May have CORS issues with SVGs)

Double-click `preview.html` in Finder

## Option 4: GitHub Pages

Visit: https://github.com/mattjss/Logo-Carousel

---

**Note:** The Vite dev server (Option 1) is the best way to see the full React app with hot reload.
